My Projects

Retail Inventory Management System

I embarked on the development of a retail inventory system to streamline inventory management processes for businesses. The system is designed to support role-based access control, allowing Admins, Managers, Employees, and Customers to interact with the system based on their respective roles.
My aim was to ensure seamless table connectivity, provide intuitive user registration, and manage stock levels effectively. Additionally, I incorporated Bootstrap and CSS for a visually appealing and responsive interface, alongside PHP and MySQL for robust backend functionality.

SQLI Environment

A development and testing of a purposefully vulnerable web application designed to demonstrate SQL injection vulnerabilities. SQL injection remains one of the most critical web application security risks, ranked third in the OWASP Top 10 2021. The application serves as an educational tool to understand how SQL injection attacks work, their potential impact, and the importance of implementing proper security measures.

Note: This application and all attack vectors presented are for educational purposes only and should never be implemented in production environments.
